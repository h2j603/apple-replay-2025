# CLAUDE.md — Apple Music Replay 2025

> 클라우드/로컬 세션 공통 컨텍스트.

## 프로젝트

2025년 가장 많이 들은 노래 인터랙티브 시각화. 바닐라 HTML/JS 단일 페이지.

- `index.html` + `app.js` — 메인 (3D LP 플레이어 인터랙션 포함)
- `data/` — 청취 데이터
- `covers/` — 앨범 커버 이미지
- `scripts/` — 데이터 가공 스크립트

빌드 없음 — 정적 파일 그대로. 브라우저로 `index.html` 열면 동작.

## ⚠️ 브랜치 주의

`backup/local-2026-06-12` 브랜치에 2026-06-12 로컬 백업 작업이 있다.
`main`과 갈라져 있으니(원격 main이 더 최신) 이 리포를 다시 만질 때
**백업 브랜치의 변경을 main과 비교·병합하는 것부터** 시작할 것.

## 전역 규칙 (모든 세션 공통)

- 대화·커밋 메시지·코드 주석 모두 **한국어**. 커밋은 명령형.
- 작업은 feature 브랜치 → `main` PR 생성 → 머지까지 완료.
- 디자인: 그림자·글로우·글래스모피즘 금지, 위계는 크기+굵기로.
  상세 → `.claude/rules/design.md`
- 작동하는 코드를 "개선" 명목으로 구조 바꾸지 않기.
